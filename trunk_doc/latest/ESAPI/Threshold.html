Class Threshold

(line 329)

Description




	Located in File: /SecurityConfiguration.php





Models a simple threshold as a count and an interval, along with a set of  actions to take if the threshold is exceeded.


These thresholds are used to define when the accumulation of a particular  event has met a set number within the specified time period. Once a threshold  value has been met, various actions can be taken at that point.
