		sanitize (line 166)
	
 
	
	

The method is similar to getSafe except that it returns a harmless value


that may or may not have any similarity to the original input (in some  cases you may not care). In most cases this should be the same as the  getSafe method only instead of throwing an exception, return some default  value.
