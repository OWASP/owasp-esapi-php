Class ValidationErrorList

(line 42)

Description




	Located in File: /ValidationErrorList.php





Use this ESAPI security control to enumerate validation exceptions.


The idea behind this interface is to define a well-formed collection of  ValidationExceptions so that groups of validation functions can be  called in a non-blocking fashion.
