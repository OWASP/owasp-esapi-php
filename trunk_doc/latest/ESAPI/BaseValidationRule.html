Class BaseValidationRule

Description
Description | Descendents | Vars (details) | Methods (details)

Implements interfaces:

  • ValidationRule

Abstract BaseValidationRule implementation of the ValidationRule interface.

This abstract class does not define getValid or sanitize methods which MUST be implemented by extending classes. All other methods need not be overridden.
