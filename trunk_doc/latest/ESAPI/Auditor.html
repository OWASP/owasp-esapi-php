Interface Auditor

(line 44)

Description




	Located in File: /Auditor.php





Use this ESAPI security control to wrap your auditing functions.


The idea behind this interface is to define a set of functions that can  be used to log security events. Implementors should use a well  established logging library.

 FUNCTIONALITY =  'FUNCTIONALITY' (line 74)
 



The FUNCTIONALITY type of log event. This is the type of event that  non-security focused loggers typically log. If you are going to log your  existing non-security events in the same log with your security events,  you probably want to use this type of log event.
