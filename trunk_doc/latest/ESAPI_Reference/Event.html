Reference implementation of an Intrusion Event.


Represents the count of and times at which a user generated an event that  corresponds to a defined IntrusionDetector threshold.  The intrusion detector  stores instances of events and invokes their increment method which  determines whether the corresponding threshold has been reached.

 Method increment (line 333)
 
 

null increment(

The increment method increments the number of times this event occurred  for this user.


Each time increment is called it will decide whether or not to throw an  IntrusionException based on the supplied count and interval parameters.  If $count is exceeded within $interval seconds then the exception will be  thrown.  This implementation maintains a kind of sliding window of  timestamps so that it can track event occurrences over time.

		
Parameters

	
    
			
  • int $count: The event count that will trigger Intrusion Detection                       within the supplied interval.
    • 
			
  • int $interval: The number of seconds within which the supplied quota of                       event occurrences will trigger Intrusion Detection.
