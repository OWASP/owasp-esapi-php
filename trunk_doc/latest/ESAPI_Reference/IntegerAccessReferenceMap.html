void update(
directReferences
$directReferences)


	

Updates the access reference map with a new set of direct references, maintaining  any existing indirect references associated with items that are in the new list.


New indirect references could be generated every time, but that  might mess up anything that previously used an indirect reference, such  as a URL parameter.

		
Parameters

	
    
			
  • directReferences $directReferences: a Set of direct references to add
