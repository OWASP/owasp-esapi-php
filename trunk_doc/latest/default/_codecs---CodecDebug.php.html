CodecDebug is a singleton class to aid Codec debugging. It buffers debug

info comprising the input to a Codec encode/decode method, as single UTF-32 encoded characters, as well as the final output from the Codec method. The debug info is logged immediately before the Codec method returns its value and the buffer is cleared at that time.
