		getDirectReference (line 95)
	
 
	
	

Get the original direct object reference from an indirect reference.


Developers should use this when they get an indirect reference from a  request to translate it back into the real direct reference. If an  invalid indirect reference is requested, then an AccessControlException is  thrown.

		update (line 201)
	
 
	
	

Updates the access reference map with a new set of direct references, maintaining  any existing indirect references associated with items that are in the new list.


New indirect references could be generated every time, but that  might mess up anything that previously used an indirect reference, such  as a URL parameter.
